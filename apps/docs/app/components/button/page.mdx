# Button

A foundational interactive component for triggering actions and interactions. Built with accessibility in mind and adapts to your theme's accent color.

<CodePreview code={`import { Button } from '@kushagradhawan/kookie-ui';

<Button>Button</Button>`}>
  <div style={{ display: 'inline-block' }}>
    <Button>Button</Button>
  </div>
</CodePreview>

## API Reference

This component is polymorphic and supports rendering as different elements via the `as` prop or composing with other components via `asChild`. Based on the `button` element by default and supports common margin props.

| Prop            | Type                                                                            | Default   |
| --------------- | ------------------------------------------------------------------------------- | --------- |
| as              | `React.ElementType`                                                             | `"button"` |
| asChild         | `boolean`                                                                       | `false`   |
| size            | `"1"` \| `"2"` \| `"3"` \| `"4"`                                                | `"2"`     |
| variant         | `"classic"` \| `"solid"` \| `"soft"` \| `"surface"` \| `"outline"` \| `"ghost"` | `"solid"` |
| color           | `string`                                                                        | -         |
| highContrast    | `boolean`                                                                       | `false`   |
| radius          | `"none"` \| `"small"` \| `"medium"` \| `"large"` \| `"full"`                    | -         |
| loading         | `boolean`                                                                       | `false`   |
| fullWidth       | `boolean`                                                                       | `false`   |
| panelBackground | `"solid"` \| `"translucent"`                                                    | -         |

### Panel Background Adaptation

Button variants automatically adapt to the theme's panel background setting. The `soft`, `surface`, `outline`, and `ghost` variants use solid colors on opaque backgrounds and alpha colors with backdrop filters on translucent backgrounds.

Use the **Theme Panel** (⚙️ icon in the top right) to toggle between "Solid" and "Translucent" panel backgrounds and see how the buttons adapt.

**Solid Panel Background**: Buttons use opaque accent colors (`accent-2`, `accent-3`, etc.) for better contrast and readability.

**Translucent Panel Background**: Buttons automatically switch to alpha accent colors (`accent-a2`, `accent-a3`, etc.) with backdrop blur for beautiful glass morphism effects.

### Component-Level Panel Background Override

Override the theme's panel background setting for individual buttons using the `panelBackground` prop. This allows you to mix solid and translucent buttons within the same theme context.

<CodePreview code={`<Theme panelBackground="translucent">
  <Box position="relative" height="200px" p="4">
    <Image 
      src="https://images.unsplash.com/photo-1638742385167-96fc60e12f59"
      alt="Background"
      style={{ position: 'absolute', inset: 0, width: '100%', height: '100%', objectFit: 'cover' }}
    />
    
    <Flex align="center" justify="center" gap="3" wrap="wrap" style={{ height: '100%' }}>
      <Button variant="soft">Theme Default</Button>
      <Button variant="soft" panelBackground="solid">Force Solid</Button>
      <Button variant="soft" panelBackground="translucent">Force Translucent</Button>
    </Flex>
  </Box>
</Theme>`}>
    <Box shadow="3" style={{ borderRadius: 'var(--radius-3)', overflow: 'hidden' }}>
      <Box position="relative" height="200px" p="4">
        <Image 
          src="https://images.unsplash.com/photo-1700508671741-319c5c61f501?q=80&w=3667&auto=format&fit=crop&ixlib=rb-4.1.0&ixid=M3wxMjA3fDB8MHxwaG90by1wYWdlfHx8fGVufDB8fHx8fA%3D%3D"
          alt="Background"
          style={{ position: 'absolute', inset: 0, width: '100%', height: '100%', objectFit: 'cover', zIndex: '-1' }}
        />
        
        <Flex align="center" justify="center" gap="3" wrap="wrap" style={{ height: '100%' }}>
          <Button color="gray" size="2" variant="soft">Theme Default</Button>
          <Button color="gray" size="2" variant="soft" panelBackground="solid">Force Solid</Button>
          <Button color="gray" size="2" variant="soft" panelBackground="translucent">Force Translucent</Button>
        </Flex>
      </Box>
    </Box>
</CodePreview>

**Use Cases:**
- **Mixed Contexts**: Use solid buttons for important actions and translucent for secondary actions
- **Design Flexibility**: Override theme settings for specific design requirements
- **Accessibility**: Force solid backgrounds when better contrast is needed
- **Visual Hierarchy**: Create emphasis through different transparency levels

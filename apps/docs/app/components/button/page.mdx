# Button

A foundational interactive component for triggering actions and interactions. Built with accessibility in mind and adapts to your theme's accent color.

<CodePreview code={`import { Button } from '@kushagradhawan/kookie-ui';

<Button>Button</Button>`}>
  <div style={{ display: 'inline-block' }}>
    <Button>Button</Button>
  </div>
</CodePreview>

## API Reference

This component is polymorphic and supports rendering as different elements via the `as` prop or composing with other components via `asChild`. Based on the `button` element by default and supports common margin props.

| Prop         | Type                                                                            | Default   |
| ------------ | ------------------------------------------------------------------------------- | --------- |
| as           | `React.ElementType`                                                             | `"button"` |
| asChild      | `boolean`                                                                       | `false`   |
| size         | `"1"` \| `"2"` \| `"3"` \| `"4"`                                                | `"2"`     |
| variant      | `"classic"` \| `"solid"` \| `"soft"` \| `"surface"` \| `"outline"` \| `"ghost"` | `"solid"` |
| color        | `string`                                                                        | -         |
| highContrast | `boolean`                                                                       | `false`   |
| radius       | `"none"` \| `"small"` \| `"medium"` \| `"large"` \| `"full"`                    | -         |
| loading      | `boolean`                                                                       | `false`   |
| fullWidth    | `boolean`                                                                       | `false`   |

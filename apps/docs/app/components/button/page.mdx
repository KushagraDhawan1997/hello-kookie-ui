# Button

Trigger an action or event, such as submitting a form or displaying a dialog.

<CodePreview code={`<Button>Click me</Button>`}>
  <div style={{ display: 'inline-block' }}>
    <Button>Click me</Button>
  </div>
</CodePreview>

## Installation

```
npm install @kushagradhawan/kookie-ui
```

## Usage

```jsx
import { Button } from '@kushagradhawan/kookie-ui';

<Button variant="outline">Button</Button>;
```

## API Reference

This component is based on the `button` element and supports common margin props.

| Prop         | Type                                                                            | Default   | Description              |
| ------------ | ------------------------------------------------------------------------------- | --------- | ------------------------ |
| size         | `"1"` \| `"2"` \| `"3"` \| `"4"`                                                | `"2"`     | Size of the button       |
| variant      | `"classic"` \| `"solid"` \| `"soft"` \| `"surface"` \| `"outline"` \| `"ghost"` | `"solid"` | Visual style variant     |
| color        | `string`                                                                        | -         | Accent color             |
| highContrast | `boolean`                                                                       | `false`   | Increases color contrast |
| radius       | `"none"` \| `"small"` \| `"medium"` \| `"large"` \| `"full"`                    | -         | Border radius            |
| loading      | `boolean`                                                                       | `false`   | Shows loading spinner    |
| fullWidth    | `boolean`                                                                       | `false`   | Makes button full width  |

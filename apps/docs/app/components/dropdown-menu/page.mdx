# Dropdown Menu

A versatile contextual menu component that displays a list of actions or options. Built with full keyboard navigation support, collision detection, and the ability to handle complex content including multiline items and rich layouts.

The Dropdown Menu component adapts to your theme's accent color and provides consistent styling across different variants while maintaining accessibility standards and smooth animations.

## API Reference

The Dropdown Menu is built as a compound component with multiple parts that work together to create flexible menu experiences.

### DropdownMenu.Root

Container component that manages the dropdown state and context.

| Prop         | Type                    | Default | Description                    |
| ------------ | ----------------------- | ------- | ------------------------------ |
| defaultOpen  | `boolean`               | `false` | Default open state             |
| open         | `boolean`               | -       | Controlled open state          |
| onOpenChange | `(open: boolean) => void` | -       | Callback when open state changes |
| modal        | `boolean`               | `true`  | Whether to use modal behavior  |

### DropdownMenu.Trigger

Button element that opens the dropdown menu when activated.

| Prop    | Type        | Default | Description                    |
| ------- | ----------- | ------- | ------------------------------ |
| asChild | `boolean`   | `false` | Compose into child element     |

### DropdownMenu.Content

The container for dropdown menu items with positioning and styling.

| Prop         | Type                           | Default  | Description                  |
| ------------ | ------------------------------ | -------- | ---------------------------- |
| size         | `"1"` \| `"2"`                 | `"2"`    | Size of the menu content     |
| variant      | `"solid"` \| `"soft"`          | `"solid"` | Visual style variant         |
| color        | `string`                       | -        | Accent color                 |
| highContrast | `boolean`                      | `false`  | Increases color contrast     |
| align        | `"start"` \| `"center"` \| `"end"` | `"start"` | Alignment relative to trigger |
| side         | `"top"` \| `"right"` \| `"bottom"` \| `"left"` | `"bottom"` | Side to position relative to trigger |
| sideOffset   | `number`                       | `4`      | Distance from the trigger    |

### DropdownMenu.Item

Individual menu item that can be activated by clicking or keyboard navigation.

| Prop     | Type      | Default | Description                |
| -------- | --------- | ------- | -------------------------- |
| color    | `string`  | -       | Accent color               |
| shortcut | `string`  | -       | Keyboard shortcut display  |
| disabled | `boolean` | `false` | Whether item is disabled   |
| asChild  | `boolean` | `false` | Compose into child element |

### DropdownMenu.CheckboxItem

Menu item with checkbox functionality for toggleable options.

| Prop     | Type                       | Default | Description                    |
| -------- | -------------------------- | ------- | ------------------------------ |
| checked  | `boolean` \| `"indeterminate"` | -       | Checked state                  |
| onCheckedChange | `(checked: boolean) => void` | -       | Callback when checked changes  |
| color    | `string`                   | -       | Accent color                   |
| shortcut | `string`                   | -       | Keyboard shortcut display      |
| disabled | `boolean`                  | `false` | Whether item is disabled       |

### DropdownMenu.RadioGroup & DropdownMenu.RadioItem

Components for creating radio button groups within the dropdown menu.

| Prop (RadioGroup) | Type                    | Default | Description                    |
| ----------------- | ----------------------- | ------- | ------------------------------ |
| value             | `string`                | -       | Selected value                 |
| onValueChange     | `(value: string) => void` | -       | Callback when value changes    |

| Prop (RadioItem) | Type      | Default | Description                |
| ---------------- | --------- | ------- | -------------------------- |
| value            | `string`  | -       | Value for this radio item  |
| color            | `string`  | -       | Accent color               |
| disabled         | `boolean` | `false` | Whether item is disabled   |

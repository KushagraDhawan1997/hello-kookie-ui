# Image

A powerful image component designed for modern web applications. Features advanced loading states, blur effects, and responsive behavior while maintaining optimal performance and accessibility.

The Image component provides smooth loading transitions, placeholder support, and skeleton states to eliminate jarring progressive loading effects. It supports multiple variants and object-fit options for flexible image display.

<CodePreview code={`import { Image } from '@kushagradhawan/kookie-ui';

<Image 
  src="${demoImages.primary}"
  alt="Demo image"
  width="300"
  height="200"
/>`}>
  <div style={{ display: 'inline-block' }}>
    <Image 
      src={demoImages.primary}
      alt="Demo image"
      width="300"
      height="200"
    />
  </div>
</CodePreview>

## API Reference

This component is based on the `img` element and supports common margin, width, and height props.

| Prop         | Type                                                                   | Default     | Description                                |
| ------------ | ---------------------------------------------------------------------- | ----------- | ------------------------------------------ |
| variant      | `"surface"` \| `"blur"`                                                | `"surface"` | Visual style variant                       |
| fit          | `"cover"` \| `"contain"` \| `"fill"` \| `"scale-down"` \| `"none"`     | `"cover"`   | How image should resize to fit container   |
| radius       | `"none"` \| `"small"` \| `"medium"` \| `"large"` \| `"full"`           | -           | Border radius                              |
| shadow       | `"1"` \| `"2"` \| `"3"` \| `"4"` \| `"5"` \| `"6"`                     | -           | Box shadow                                 |
| loading      | `"eager"` \| `"lazy"`                                                  | `"lazy"`    | Loading behavior                           |
| placeholder  | `string`                                                               | -           | Placeholder image URL while loading        |
| showSkeleton | `boolean`                                                              | `false`     | Shows skeleton placeholder while loading   |
| onLoad       | `(event: React.SyntheticEvent<HTMLImageElement>) => void`              | -           | Callback when image loads                  |
| onError      | `(event: React.SyntheticEvent<HTMLImageElement>) => void`              | -           | Callback when image fails to load          |
| alt          | `string`                                                               | -           | Alternative text (required for accessibility) |
| asChild      | `boolean`                                                              | `false`     | Merge props into immediate child element   |

## [1.0.0] - 2024-01-15

### Added
- Initial release of IconButton component with comprehensive icon-only functionality
- Four size options (1, 2, 3, 4) with square aspect ratios
- Six visual variants: solid, outline, soft, ghost, surface, classic
- Comprehensive color system with semantic options (blue, mint, orange, crimson, gray)
- Full accessibility support with keyboard navigation and screen reader compatibility
- Loading states with visual feedback and ARIA announcements
- Disabled states with proper ARIA attributes and visual indicators
- Polymorphic rendering with `as` prop and `asChild` support
- High contrast mode for enhanced visibility
- Responsive design with breakpoint support
- Flush behavior for seamless text integration
- Tooltip system with customizable positioning and timing
- Form integration with submit and reset functionality
- Panel background support for both solid and translucent materials
- Enhanced accessibility features with mandatory labeling requirements
- Automatic icon sizing system that scales with button size
- Square aspect ratios for consistent visual alignment in toolbars and grids
- Built-in tooltip integration for icon-only interactions
- Advanced tooltip positioning and timing controls
- Material support for both solid and translucent interfaces

### Changed
- Systematic icon sizing that follows Button typography scale (12px → 16px → 18px → 20px)
- Improved visual rhythm and consistency across all interface elements
- Enhanced integration with overall design system tokens
- Better accessibility compliance with WCAG 2.1 standards

### Fixed
- Runtime validation of accessibility requirements with development warnings
- Enhanced ARIA attribute support for screen reader compatibility
- Automatic icon size application through CSS custom properties
- Prevention of manual icon sizing that could break visual consistency
- Improved focus management and keyboard navigation
- Better touch target sizing for mobile accessibility

### Technical
- Built on Radix UI primitives for robust accessibility foundation
- CSS custom properties for comprehensive theming support
- TypeScript support with comprehensive type definitions
- Design token integration for consistent spacing, sizing, and colors
- Systematic color contrast calculations meeting WCAG standards
- Motion preference support for users with vestibular disorders
- Touch target optimization for mobile and accessibility compliance

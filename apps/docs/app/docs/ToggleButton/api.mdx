# API

The ToggleButton component extends Button with toggle state management for on/off interactions. It provides controlled and uncontrolled state handling, visual feedback for pressed states, and maintains all Button functionality.

For comprehensive documentation on shared features like variants, sizes, colors, states, and other Button functionality, see the **[Button API documentation](../Button/api)**.

## Import

```tsx
import { ToggleButton } from '@kushagradhawan/kookie-ui';
```

## Props

ToggleButton inherits all Button props and adds toggle-specific state management props.

### Pressed State

ToggleButton adds a pressed state that provides visual feedback for on/off interactions.

```tsx
{/* Pressed state - on */}
<ToggleButton size="3" variant="soft" pressed={true}>
  Enable Feature
</ToggleButton>

<ToggleButton size="3" variant="soft" pressed={false}>
  Disable Feature
</ToggleButton>
```

# ToggleButton

Use toggle buttons for stateful interactions that remember their on/off position. Toggle buttons provide clear visual feedback and predictable toggle behavior for actions that can be enabled or disabled.

Choose from six visual variants to match your interface context, four sizes for different interface densities, and comprehensive color options for semantic communication. The component automatically handles state management, provides controlled and uncontrolled patterns, and maintains all Button functionality with enhanced toggle capabilities.

For interfaces that need to feel elevated above their background context, use the `material` prop to create depth and establish visual hierarchy through translucency.

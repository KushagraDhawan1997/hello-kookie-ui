# API

The ToggleIconButton component extends IconButton with toggle state management for icon-only on/off interactions. It provides controlled and uncontrolled state handling, visual feedback for pressed states, and maintains all IconButton functionality.

For comprehensive documentation on shared features like variants, sizes, colors, states, and other Button functionality, see the **[Button API documentation](../Button/api)**.

## Import

```tsx
import { ToggleIconButton } from '@kushagradhawan/kookie-ui';
```

## Props

ToggleIconButton inherits all IconButton props and adds toggle-specific state management props.

### Pressed State

ToggleIconButton adds a pressed state that provides visual feedback for on/off interactions.

<CodeBlock
  preview={
    <Flex gap="2" align="center">

        <Bold />

      <ToggleIconButton size="3" variant="soft" pressed={false} aria-label="Italic text">
        <Italic />
      </ToggleIconButton>
    </Flex>
  }

```tsx
{/* Pressed state - on */}
<ToggleIconButton size="3" variant="soft" pressed={true} aria-label="Bold text">
  <Bold />
</ToggleIconButton>

{/* Pressed state - off */}
<ToggleIconButton size="3" variant="soft" pressed={false} aria-label="Italic text">
  <Italic />
</ToggleIconButton>
```

</CodeBlock>

## Icons

ToggleIconButton automatically sizes icons based on the button size. Never manually size icons in ToggleIconButton.

<CodeBlock
  preview={
    <Flex gap="2" align="center">

      </ToggleIconButton>

        <Bold />

  }
>

```tsx
{/* Icons are automatically sized */}
<ToggleIconButton size="1" variant="soft" aria-label="Settings">
  <Bold />
</ToggleIconButton>

{/* Don't manually size icons */}
<ToggleIconButton size="2" variant="soft" aria-label="Settings">
  <Bold size={16} /> {/* ❌ Don't do this */}
</ToggleIconButton>
```

# ToggleIconButton

Use toggle icon buttons for compact, stateful interactions that remember their on/off position. Toggle icon buttons provide efficient interaction patterns for toolbars, navigation elements, and utility functions where space is limited and visual clarity is essential.

Choose from six visual variants to match your interface context, four sizes for different interface densities, and comprehensive color options for semantic communication. The component automatically handles state management, icon sizing, enforces accessibility requirements, and provides built-in tooltip integration for enhanced usability.

For interfaces that need to feel elevated above their background context, use the `material` prop to create depth and establish visual hierarchy through translucency.

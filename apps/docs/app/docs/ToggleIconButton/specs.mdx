# Specs

The ToggleIconButton system extends Button with toggle state management and icon-specific specifications. It uses the same mathematical precision and design tokens as Button but adds square aspect ratios, automatic icon sizing, and toggle state indicators for icon-only on/off interactions.

ToggleIconButton inherits most specifications from Button. This documentation focuses on what's unique to ToggleIconButton.

## Typography

ToggleIconButton does not use typography since it contains only icons. For typography specifications, see the **[Button Specs documentation](../Button/specs)**.

## Spacing

ToggleIconButton uses the same spacing system as Button but with square aspect ratios. For comprehensive spacing specifications, see the **[Button Specs documentation](../Button/specs)**.

### Square Dimensions

ToggleIconButton maintains a perfect square aspect ratio (height = width) regardless of content. This ensures consistent visual alignment in toolbars, grids, and compact interfaces where rectangular buttons would create visual noise.

**Square Aspect Ratio:** ToggleIconButton maintains a perfect square aspect ratio (height = width) regardless of content. This ensures consistent visual alignment in toolbars, grids, and compact interfaces where rectangular buttons would create visual noise.

### Automatic Icon Sizing

ToggleIconButton automatically sizes icons based on the button size. The component handles all icon scaling internally to maintain visual consistency and prevent manual sizing errors.

### Toggle State Management

ToggleIconButton adds toggle state management that provides visual feedback for on/off interactions.

**State Communication:**
- **Pressed state:** Visual indication that toggle is active/on
- **Unpressed state:** Visual indication that toggle is inactive/off
- **State transitions:** Smooth visual feedback when toggling between states
- **Accessibility:** Automatic ARIA attribute management for screen readers

**Toggle Implementation:**
- Built on Radix UI Toggle primitive for robust state management
- Seamless integration with IconButton component through asChild pattern
- Automatic `aria-pressed` attribute management based on toggle state
- Enhanced keyboard navigation support for toggle interactions

# Kookie UI
A design system for information-dense SaaS, built on Radix infrastructure. Most design systems optimize for marketing sites. Kookie UI is built for complex, data-heavy interfaces.

Started as a fork of [Radix Themes](https://www.radix-ui.com/themes) in June 2024. Built on [Radix Primitives](https://www.radix-ui.com/primitives) infrastructure. Adds visual design optimized for information density and components like Sidebar, Calendar, Data Tables, Command Palette.
